wiki Phasmophobia

Presentation rapide

Phasmophobia est un jeu de survie developpe par un studio independant, Kinetic Games. Il est sorti en septembre 2020. Le but de ce jeu est de decouvrir dans un lieu hante, le type de fantome qui nous veut du mal sans se faire tuer. Il existe plusieurs modes de difficultes: amateur, intermediaire, expert, cauchemar, personnalise. Vous pouvez rencontrer plusieurs types de fantomes avec des specificites differentes que vous devrez chasser avec des outils speciaux, a vous de choisir quel outil sera le plus adapte dans chaque situation.
