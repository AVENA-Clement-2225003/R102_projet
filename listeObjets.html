Appareil photo

  • Prix: 40$
  • Niveau d'achat: 0
  • Utilisation: 5 photos maximum, clique droit pour prendre une photo.

Cet equipement permet de photographier differentes interations afin d'accomplir certains objectifs. Les photos peuvent rapportes de l'argent selon leurs etoiles.

Bougie

  • Prix: 15$
  • Niveau d'achat: 0
  • Utilisation: illimite

Permet d'eclairer une zone autour d'un jouer permettant ainsi de conserver sa sante mentale. Doit etre allumee grace a un briquet.

Camera

  • Prix: 50$
  • Niveau d'achat: 0
  • Utilisation: illimite

Permet d'observer les orbes fantomatiques dans une piece depuis le camion. La camera possede deux modes: un mode "normal" et un mode nocturne. C'est ce mode nocturne qui permet d'observer ces orbes. Ces dernieres constituent une preuve dans le journal.

Camera Frontale

  • Prix: 60$
  • Niveau d'achat: 0
  • Utilisation: illimite

La camera frontale permet d'afficher le point de vue du joueur. Point de vu visible depuis le camion. Permet egalement d'observer les orbes fantomatiques.

Crucifix

  • Prix: 30$
  • Niveau d'achat: 0
  • Utilisation: le placer dans la piece ou se trouve l'entite

Le crucifix permet d'annuler la periode de chasse de l'entite si il est place dans sa piece.

Detecteur de mouvement

  • Prix: 100$
  • Niveau d'achat: 0
  • Utilisation: illimite

Le detecteur de mouvement permet de detecter si un joueur ou une entite passe a proximiter. Dans ce cas-la, il s'eclaire.

EMF

  • Prix: 45$
  • Niveau d'achat: 0
  • Utilisation: illimite

L'EMF est un equipement qui reagit a la presence de l'entite et aux interactions de cette derniere dans la piece. Peut constituer une preuve dans le journal.

Encens

  • Prix: 15$
  • Niveau d'achat: 0
  • Utilisation: une seule fois, actif pendant 14 secondes

L'encens s'allume a l'aide d'un briquet. Il laisse une trainee d'encens et permet de proteger le joueur d'une chasse avec l'entite.

Lampe torche

  • Prix: 30$ et 50$ pour la puissante
  • Niveau d'achat: 0
  • Utilisation: illimitee

La lampe torche permet d'eclairer les zones obscures. Existe en version amelioree.

Lampe UV

  • Prix: 35$
  • Niveau d'achat: 0
  • Utilisation: illimitee

La lampe UV permet de reveler les empreintes que laisse l'entite derriere lui. Il est donc utile de regarder les poignets de porte, interrupteurs ou fenetre avec. Elle permet de constituer une preuve.

Livre

  • Prix: 40$
  • Niveau d'achat: 0
  • Utilisation: une fois

Le livre d'ecriture phantomatique permet a l'entite de laisser une trace, un message ou une ecriture. Il faut le placer dans la piece d'affectation de l'entite pour qu'il puisse le trouver. Il peut constituer une preuve dans le journal.

Microphone parabolique

  • Prix: 50$
  • Niveau d'achat: 0
  • Utilisation: illimite

Le microphone parabolique enregistre tout les sons emis par l'entite en fonction de l'endroit ou le curseur est place. Des valeurs sont affichees sur un petit ecran. Plus le son es grand, plus les valeurs sont grandes et inversement.

Pillules

  • Prix: 45$
  • Niveau d'achat: 0
  • Utilisation: une seule fois

Les pillules de sante permettent de retablir la sante mentale si elle est consommee.

Projecteur DOTS

  • Prix: 65$
  • Niveau d'achat: 0
  • Utilisation: illimitee

Le Projecteur D.O.T.S. projette des petits points verts sur une zone precise. Il permet de reveler la presence de l'entite si ce dernier passe devant le projecteur. Il peut constituer une preuve dans le journal

Sel

  • Prix: 15$
  • Niveau d'achat: 0
  • Utilisation: 3 poses par boites

Le sel, lorsqu'il est pose au sol, permet de marquer les pas de l'entite lorsqu'elle marche dessus.

Spiritbox

  • Prix: 50$
  • Niveau d'achat: 0
  • Utilisation: illimite

La spiritbox est un outil qui permet de communiquer avec l'entite. Peut constituer une preuve dans le journal

Thermometre

  • Prix: 30$
  • Niveau d'achat: 0
  • Utilisation: illimite

Le thermometre permet d'enregistrer la temperature a l'endroit ou est place le curseur. Si la tempretature est en dessous de 12 degres l'entite est dans la piece. Si la temperature est negaive, elle peut constituer une preuve dans le journal

Boite a Musique

  • Difficultes: 1/3
  • Danger: 2/3
  • Aide aux preuves: 2/4

La boite a musique fait perdre, lors de son utilisation, une grosse quantite de sante mentale et force l'entite a chanter. Attention a ne pas vous approcher de l'entite avec!

Carte de tarot

  • Difficultes: 2/3
  • Danger: 3/3
  • Aide aux preuves: 2/4
  • Le Soleil -> Restaure la sante mentale a 100%
  • La Lune -> La sante mentale le tombe a 100%
  • La Tour -> Cause une interaction
  • La Roue de la fortune -> Augmente la sante mental de 25% ou alors la baisse d'autant selon la couleur de la flamme
  • Le Diable -> Declenche un evenement avec l'entite
  • La Grande pretresse -> Un coequipier choisi aleatoirement est ressucite
  • Le Pendu -> Le joueur est tue instantanement
  • La Mort -> Cause une chasse
  • L'Hermite -> Oblige l'entite a rester dans sa piece pendant un court moment
  • Le Fou -> Cette carte n'a aucun effet

Chaque jeu de tarot contient au total 10 cartes generees aleatoirement parmi 10 modeles integres dans le jeu. Des effets positifs ou negatifs peuvent survenir a chacun des tirages. Une meme carte peut apparaitre plusieurs fois et les certaines cartes sont plus rares a optenir que d'autres. Voici la liste de ces cartes:

Cercle invocation

  • Difficultes: 3/3
  • Danger: 3/3
  • Aide aux preuves: 3/4

Pour pouvoir l'utiliser il faut allumer les 5 bougies rouges avec un briquet afin de faire apparaitre l'entite au milieu. l'entite sera immobiliser pendant 4 secondes avant de commencer une chasse.

Planche OUIJA

  • Difficultes: 1/3
  • Danger: 1/3
  • Aide aux preuves: 2/4

La planche de ouija permet de poser des questions a l'entite. Poser des questions a l'entite fait perdre de la sante mentale. Il est important de dire au revoir a l'entite a la fin de l'interaction...

Poupee vaudou

  • Difficultes: 1/3
  • Danger: 2/3
  • Aide aux preuves: 3/4

En interragissant avec la poupee vaudou, une des 10 epingles a disposition va s'enfoncer dans corps vous perdrez a chaque fois 10% de votre sante mentale mais en contre partie une interaction avec l'entite aura lieu. Si vous enfoncez une aiguille dans le coeur ou si vous avez moins de 10% de sante mentale, une chasse se lance.
